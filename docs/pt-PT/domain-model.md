# Domain Model

## Entidades Principais

### Gym
Representa um ginásio.

### Member
Representa um membro do ginásio.

### Lead
Representa um potencial cliente.

### Conversation
Representa uma conversa WhatsApp.

### Message
Representa uma mensagem.

### Campaign
Representa uma campanha de fidelização.

### CampaignMessage
Representa uma mensagem de campanha.

### FAQ
Base de conhecimento.

### Plan
Plano de adesão.

### Promotion
Promoção ativa.
