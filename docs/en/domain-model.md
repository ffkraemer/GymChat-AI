# Domain Model

## Core Entities

### Gym
Represents a gym.

### Member
Represents a gym member.

### Lead
Represents a potential customer.

### Conversation
Represents a WhatsApp conversation.

### Message
Represents a message.

### Campaign
Represents a loyalty campaign.

### CampaignMessage
Represents a campaign message.

### FAQ
Knowledge base entity.

### Plan
Membership plan.

### Promotion
Active promotion.
